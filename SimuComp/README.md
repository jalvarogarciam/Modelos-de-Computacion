## ¿Qué es SIMUCOMP?

**SIMUCOMP** es una herramienta educativa portable para explorar y comparar los modelos clásicos de computabilidad. Su propósito es servir como **simulador educativo** de escritorio que permita:

- Ejecutar programas en **GOTO**, **LOOP**, **WHILE** y **URM**  
- Comparar visualmente su comportamiento y **equivalencia**  
- Crear y gestionar **macros propias**
- Ejecutar en modo completo o **paso a paso** con **traza detallada**  
- Visualizar registros y estados en tiempo real  

## Uso del simulador

El repositorio contiene:

- **SimuComp.jar** → Versión ejecutable autocontenida (no requiere instalación).  
  Para usarlo, tienes dos opciones:  
  1. **Desde la terminal:** ubícate en la carpeta donde descargaste el archivo y ejecuta:  
     ```bash
     java -jar [Dirección donde esté el simulador]/SimuComp.jar
     ```
  2. **Doble clic:** si tu sistema tiene Java instalado, también puedes abrir el simulador simplemente haciendo doble clic sobre el archivo `.jar`.
